# 日历/倒计时

## 📅 核心功能

- ✅ **日期查询** - 获取星期几
- ✅ **天数计算** - 计算相隔天数
- ✅ **倒计时** - 重要日子倒数
- ✅ **重要日期** - 管理生日/纪念日
- ✅ **日历显示** - 查看月历

## 📖 使用方法

### 查询星期

```
2026 年 12 月 25 日是星期几
今天是星期几
```

**脚本调用：**
```bash
python scripts/calendar.py weekday 2026-12-25
python scripts/calendar.py weekday
```

### 计算相隔天数

```
距离 2026 年春节还有多少天
2026 年 1 月 1 日到现在多少天
```

**脚本调用：**
```bash
python scripts/calendar.py days 2026-01-01
```

### 倒计时

```
距离圣诞节还有多少天
距离生日倒计时
```

**脚本调用：**
```bash
python scripts/calendar.py countdown 2026-12-25 圣诞节
```

### 管理重要日期

```
添加重要日期：2026-12-25 圣诞节
添加每年重复的日期：2026-05-20 纪念日 --recurring
查看重要日期
```

**脚本调用：**
```bash
python scripts/calendar.py add 2026-12-25 圣诞节
python scripts/calendar.py add 2026-05-20 纪念日 --recurring
python scripts/calendar.py list
```

### 查看日历

```
显示 2026 年 12 月的日历
```

**脚本调用：**
```bash
python scripts/calendar.py calendar 2026 12
```

## 📊 输出示例


### 倒计时
```
⏳ 圣诞节倒计时
目标：2026-12-25

```

### 重要日期
```

🎉 2026-03-11：今天 - 就是今天！
🔄 2026-05-20：纪念日 - 还有 70 天
2026-12-25：圣诞节 - 还有 288 天
```

## 💡 使用技巧

- 用 countdown 设置目标倒计时
- 重要日期支持每年重复
- 自动显示已过/即将到来
- 支持查看任意年月日历
