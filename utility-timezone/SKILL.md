# 时区转换工具

## 🌍 核心功能

- ✅ **时区转换** - 任意时区间转换
- ✅ **当前时间** - 查询世界各地当前时间
- ✅ **时区列表** - 查看支持的时区
- ✅ **周末识别** - 自动判断是否周末

## 📖 使用方法

### 时区转换

```
北京时间 12 点是纽约几点
```

**脚本调用：**
```bash
python scripts/timezone.py convert "2026-03-11 12:00" Beijing NewYork
```

### 查询当前时间

```
东京现在几点
```

**脚本调用：**
```bash
python scripts/timezone.py now Tokyo
```

### 列出时区

```
支持哪些时区
```

**脚本调用：**
```bash
python scripts/timezone.py list
```

## 💡 使用技巧

- 支持城市名：Beijing, Tokyo, NewYork 等
- 支持时区偏移：+8, -5, +05:30
- 自动处理跨天情况
- 识别周末方便安排会议
