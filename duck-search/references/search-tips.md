# DuckDuckGo 搜索技巧

## 🔍 搜索语法

DuckDuckGo 支持这些搜索语法：

### 精确匹配
```
"exact phrase"     # 精确匹配短语
```

### 排除词汇
```
python -tutorial   # 搜索 python 但排除 tutorial
```

### 站内搜索
```
site:github.com    # 只在 github.com 内搜索
```

### 文件类型
```
filetype:pdf       # 只搜索 PDF 文件
```

### 标题搜索
```
intitle:React      # 标题中包含 React
```

## 💡 关键词优化

### 技术搜索
- 加版本号：`Next.js 15` 而不是 `Next.js`
- 加年份：`best AI tools 2025`
- 用英文：`Python tutorial` 比 `Python 教程` 结果多

### 常见问题
- 加 `how to`：`how to install Python`
- 加 `vs` 对比：`Cursor vs VS Code`
- 加 `tutorial` 找教程：`React hooks tutorial`

## ⚡ 高级技巧

### 组合语法
```
"async python" site:realpython.com -beginner
```

### 查找代码
```
python async example filetype:ipynb
```

### 查找文档
```
intitle:documentation React hooks
```

## 📊 结果筛选

搜索后手动筛选：
1. 优先看官方文档
2. 看日期（技术内容选新的）
3. 避开广告和软文
4. 多来源交叉验证
