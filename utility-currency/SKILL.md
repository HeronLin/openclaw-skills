# 汇率转换工具

## 💱 核心功能

- ✅ **货币转换** - 支持 29 种货币
- ✅ **汇率查询** - 查看对 USD/CNY 汇率
- ✅ **货币列表** - 查看所有支持的货币
- ✅ **离线使用** - 无需网络连接

## 📖 使用方法

### 货币转换

```
100 美元等于多少人民币
1000 人民币换多少美元
```

**脚本调用：**
```bash
python scripts/currency.py convert 100 USD CNY
python scripts/currency.py convert 1000 CNY USD
```

### 查询汇率

```
欧元的汇率是多少
```

**脚本调用：**
```bash
python scripts/currency.py rate EUR
```

### 列出货币

```
支持哪些货币
```

**脚本调用：**
```bash
python scripts/currency.py list
```

## 💡 使用技巧

- 支持 29 种全球主要货币
- 自动识别货币代码（不区分大小写）
- 汇率为近似值，仅供参考
- 实际交易请以银行汇率为准
