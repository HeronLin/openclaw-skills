# 随机决策工具

## 🎲 核心功能

- ✅ **掷骰子** - 支持任意面数
- ✅ **抛硬币** - 正反面随机
- ✅ **抽签** - 从选项中随机选择
- ✅ **随机数** - 生成指定范围的随机数
- ✅ **打乱顺序** - 随机打乱列表

## 📖 使用方法

### 掷骰子

```
掷个骰子
掷 3 个 20 面骰子
```

**脚本调用：**
```bash
python scripts/random.py dice
python scripts/random.py dice 20 3
```

### 抛硬币

```
抛硬币
抛 5 次硬币
```

**脚本调用：**
```bash
python scripts/random.py coin
python scripts/random.py coin 5
```

### 抽签/随机选择

```
帮我决定：吃饭，睡觉，打游戏
从苹果、香蕉、橙子中选一个
```

**脚本调用：**
```bash
python scripts/random.py draw "吃饭，睡觉，打游戏"
python scripts/random.py draw "苹果，香蕉，橙子"
```

### 生成随机数

```
生成 1 到 100 的随机数
生成 5 个不重复的随机数
```

**脚本调用：**
```bash
python scripts/random.py number 1 100
python scripts/random.py number 1 100 5
```

### 打乱顺序

```
打乱这些顺序：A,B,C,D,E
```

**脚本调用：**
```bash
python scripts/random.py shuffle "A,B,C,D,E"
```


### 抛硬币
```

```

### 抽签
```

选项：吃饭，睡觉，打游戏

```

```

```

## 💡 使用技巧

- 掷骰子支持任意面数（4/6/8/10/12/20 等）
- 抽签用于解决选择困难症
- 随机数可用于抽奖、游戏
- 打乱顺序用于随机排序
