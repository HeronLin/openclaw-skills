# 天气查询技能

## 🌤️ 核心特点

- ✅ **完全免费** - 无需 API Key
- ✅ **全球支持** - 支持任何城市
- ✅ **中文输出** - 自动识别语言
- ✅ **简单快速** - 10 秒内返回结果

## 📖 使用方法

### 基本查询

```
"北京天气怎么样"
"上海今天天气"
"深圳温度"
```

### 脚本调用

```bash
python scripts/weather.py 北京
```

### 输出示例

```
🌤️ 北京 天气

🌡️ 温度：15°C (体感 13°C)
☁️ 天气：晴
💧 湿度：45%
💨 风力：西北 10km/h

📅 未来 3 天预报:
  2026-03-11: 8°C ~ 18°C
  2026-03-12: 10°C ~ 20°C
  2026-03-13: 12°C ~ 22°C
```

## 🌍 支持的城市

任何城市名称都可以：
- 中文：北京、上海、深圳
- 英文：Beijing、Shanghai、Shenzhen
- 国际：New York、London、Tokyo

## 💡 使用技巧

- 直接说城市名 + 天气
- 支持区县级：南山区、浦东新区
- 支持景区：黄山、西湖
