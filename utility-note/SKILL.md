# 笔记/备忘录

## 📝 核心功能

- ✅ **添加笔记** - 支持标题、内容、标签
- ✅ **查看笔记** - 按 ID 查看
- ✅ **搜索笔记** - 全文搜索
- ✅ **标签管理** - 分类整理
- ✅ **笔记分析** - 统计使用情况

## 📖 使用方法

### 添加笔记

```
添加笔记：购物清单 牛奶、鸡蛋、面包
添加笔记：会议记录 今天讨论了项目进度 --tags 工作，会议
```

**脚本调用：**
```bash
python scripts/note.py add "购物清单" "牛奶、鸡蛋、面包"
python scripts/note.py add "会议记录" "今天讨论了项目进度" --tags 工作，会议
```

### 查看笔记

```
查看笔记 1
```

**脚本调用：**
```bash
python scripts/note.py view 1
```

### 列出笔记

```
我的笔记
查看所有笔记
```

**脚本调用：**
```bash
python scripts/note.py list
python scripts/note.py list --tag 工作
```

### 搜索笔记

```
搜索笔记 会议
查找包含"项目"的笔记
```

**脚本调用：**
```bash
python scripts/note.py search "会议"
```

### 删除笔记

```
删除笔记 1
```

**脚本调用：**
```bash
python scripts/note.py delete 1
```

## 📊 输出示例

## 💡 使用技巧

- 使用标签分类（工作、生活、学习等）
- 定期搜索回顾旧笔记
- 用 analyze 查看使用情况
- 标题要简洁明了
