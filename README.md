# OpenClaw Skills Collection

🦞 龙虾军团的 OpenClaw 技能集合

## 已发布的技能

### web-search
网络搜索和信息获取技能
- 使用 Perplexity API 进行高效网络搜索
- 支持时间/语言/地区过滤
- 网页内容抓取和提取

**使用方法：**
```
npx skills install <repo-url>
```
