## 🚀 已发布的技能

### 🦆 duck-search ⭐ 推荐

**完全免费**的网络搜索技能！

**功能：**
- ✅ 无需 API Key
- ✅ 无需注册
- ✅ 开箱即用
- ✅ 隐私保护

**安装依赖：**
```bash
pip install ddgs
```

**示例：**
```
"帮我搜索 Python 教程"
"查找 AI 工具推荐"
"搜索 React hooks 文档"
```

📖 [查看安装指南](./duck-search/INSTALL.md)

---

### 🔍 web-search

专业的网络搜索技能（需要 Perplexity API Key）

**功能：**
- 使用 Perplexity API 进行高效网络搜索
- 支持时间/语言/地区过滤
- 网页内容抓取和提取为 Markdown

**示例：**
```
"帮我搜索 2025 年最好的 AI IDE"
"查找 Next.js 15 的迁移指南"
```

📖 [查看详细文档](./web-search/SKILL.md)
