# 端口占用查询

## 🔌 核心功能

- ✅ **端口检查** - 查看端口是否被占用
- ✅ **进程信息** - 显示占用进程的详细信息
- ✅ **列出端口** - 显示所有监听中的端口
- ✅ **快速诊断** - 帮助解决端口冲突

## 📖 使用方法

### 检查端口占用

```
检查 80 端口是否被占用
查看 3306 端口被谁占用
```

**脚本调用：**
```bash
python scripts/portcheck.py check 80
python scripts/portcheck.py check 3306
```

### 列出所有监听端口

```
列出所有开放的端口
```

**脚本调用：**
```bash
python scripts/portcheck.py list
```
